# Intégration d'Unidoo dans un projet

  1. Installer le module
npm i @sedoo/unidoo
  1. Dans le fichier main.js ajouter les lignes suivantes:
import Unidoo from '@sedoo/unidoo'
Vue.use(Unidoo)
